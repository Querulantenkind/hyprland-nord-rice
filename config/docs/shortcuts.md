# Hyprland Nord Rice - Keybindings

## Overview

All available keybindings in the Hyprland Nord Rice setup. `$mainMod` is the SUPER/Windows key.

## Launch Applications

| Keybinding | Action | Description |
|------------|--------|-------------|
| `$mainMod + Return` | Kitty Terminal | Opens the terminal |
| `$mainMod + E` | Thunar File Manager | Opens the file manager |
| `$mainMod + B` | Firefox Browser | Opens the web browser |
| `$mainMod + Space` | Wofi Launcher | Main launcher with all modes |
| `$mainMod + D` | Wofi Drun | Direct application launcher |
| `$mainMod + V` | Clipboard History | Shows clipboard history |
| `$mainMod + C` | Calculator | Opens calculator |
| `$mainMod + M` | Emoji Picker | Opens emoji selection |
| `$mainMod SHIFT + R` | Scripts | Opens script menu |

## Window Management

| Keybinding | Action | Description |
|------------|--------|-------------|
| `$mainMod + Q` | Close window | Closes active window |
| `$mainMod SHIFT + E` | Exit Hyprland | Exits the Hyprland session |
| `$mainMod + F` | Fullscreen | Toggles fullscreen mode |
| `$mainMod SHIFT + F` | Floating toggle | Toggles between floating/tiled |
| `$mainMod + P` | Pseudo-tile | Toggles pseudo-tile mode |
| `$mainMod + J` | Split toggle | Switches split direction |

## Advanced Window Management

| Keybinding | Action | Description |
|------------|--------|-------------|
| `$mainMod + Tab` | Hyprspace Overview | Opens macOS/GNOME-style workspace overview with drag & drop |
| `$mainMod SHIFT + Tab` | Previous window | Jumps to previous window |
| `$mainMod + W` | Window groups | Toggles window groups |
| `$mainMod + G` | Next group | Jumps to next group |
| `$mainMod SHIFT + G` | Previous group | Jumps to previous group |
| `$mainMod + Z` | Pin window | Pins window to position |

## Workspaces

| Keybinding | Action | Description |
|------------|--------|-------------|
| `$mainMod + 1-0` | Workspace 1-10 | Switches to workspace 1-10 |
| `$mainMod SHIFT + 1-0` | Move window | Moves window to workspace 1-10 |
| `$mainMod + [` | Previous workspace | Jumps to previous workspace |
| `$mainMod + ]` | Next workspace | Jumps to next workspace |

## Scratchpads & Special Workspaces

| Keybinding | Action | Description |
|------------|--------|-------------|
| `$mainMod + S` | Special workspace | Opens special workspace (Magic) |
| `$mainMod SHIFT + S` | Move to special | Moves window to special workspace |
| `$mainMod + ` | Scratchpad terminal | Opens terminal as scratchpad |
| `$mainMod SHIFT + ` | Toggle scratchpad | Shows/hides scratchpad |
| `$mainMod + .` | Scratchpad audio | Opens audio manager as scratchpad |
| `$mainMod + ,` | Scratchpad files | Opens file manager as scratchpad |

## Focus & Movement

| Keybinding | Action | Description |
|------------|--------|-------------|
| `$mainMod + ←/→/↑/↓` | Switch focus | Switches focus in direction |
| `$mainMod + H/L/K/J` | Switch focus (Vim-style) | Alternative direction keys |
| `$mainMod SHIFT + ←/→/↑/↓` | Move window | Moves window in direction |
| `$mainMod SHIFT + H/L/K/J` | Move window | Alternative movement |
| `$mainMod CTRL + ←/→/↑/↓` | Resize window | Changes window size |

## Tips

- **$mainMod** is the **SUPER/Windows** key by default
- All keybindings are defined in `~/.config/hypr/hyprland.conf`
- You can customize the keys in the configuration
- For complete documentation: `$mainMod + H` → Documentation

## Search & Navigation

- Press `$mainMod + Space` for the main launcher
- Select "Documentation" for this overview
- Or press `$mainMod + H` directly for quick access
